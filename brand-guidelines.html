Light Mode Variant

For LinkedIn posts, email clients, and print materials. Light mode often performs better on LinkedIn where the feed is predominantly light-themed.

When to Use Dark

Instagram, Twitter/X, Stories, tech-focused audiences. Dark stands out in feeds and feels more premium/modern.

When to Use Light

LinkedIn, email newsletters, embedded content on light websites. Feels native to the platform and more approachable.

Typography

Keep it simple. System fonts render reliably across all devices and feel professional at any size.

Headlines — Inter / System UI / -apple-system
AI Evals For Engineers & PMs
Body Text — Same family, regular weight
"I found this course super useful and comprehensive. I now have a significantly better way to measure and evaluate LLM use cases in complicated scenarios."

Headlines

Inter or system font, 600-700 weight. Use for titles, key stats, and primary messaging.

Body Copy

Same font family, 400 weight. Use for testimonial quotes, descriptions, and supporting text.

Captions

Smaller size (0.75-0.85rem), muted color (#a1a1aa). Use for attribution, metadata, and labels.

Do's and Don'ts

✓ Do

  • Use dark backgrounds (#0f1114)
  • Use amber/gold for primary accents and CTAs
  • Use real photos of Hamel & Shreya
  • Include real names, titles, and companies on testimonials
  • Leave generous whitespace—let content breathe
  • Keep typography clean and highly readable
  • Include the "Lenny's List" badge for credibility
  • Use subtle gradients and soft shadows

✗ Don't

  • Use neon green, hot pink, or "gaming" colors
  • Use more than 2 accent colors per asset
  • Use aggressive urgency ("LIMITED TIME ONLY!!!")
  • Use stock photos of generic businesspeople
  • Crowd content with too many elements
  • Use heavy emoji in professional copy
  • Use harsh drop shadows or beveled effects
  • Make it look like a coding bootcamp ad
